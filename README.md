# ImageAnnotator

Lightweight image annotation tool for yolov3.

Just click to the import button and select the folder that contains jpg images and txt files.

### Controls
1. W: MOVE UP
2. A: MOVE LEFT
3. S: MOVE DOWN
4. D: MOVE RIGHT
5. Q: SWITCH TO PREVIOUS IMAGE
6. E: SWITCH TO NEXT IMAGE
7. R: CHANGE THE BOX CLASS
8. Z: UNDO
9. SPACE: SAVE BOXES TO TXT FILE AND SWITCH TO NEXT IMAGE
