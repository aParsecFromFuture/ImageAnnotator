# ImageAnnotator

Lightweight image annotation tool for yolov3.

### Controls
1. W: MOVE UP
2. A: MOVE LEFT
3. S: MOVE DOWN
4. D: MODE RIGHT
5. Q: PREVIOUS IMAGE
6. E: NEXT IMAGE
7. R: CHANGE THE BOX CLASS
8. Z: UNDO
9. SPACE: SAVE BOXES TO TXT FILE AND SWITCH TO NEXT IMAGE
